TeXmacs plugin to move beamer slides around
------------------------------------

When this plugin is installed, two icons will appear on the focus toolbar of 
Beamer documents in paper or panorama view. These buttons allow to move slides
up and down in the presentation.

![](screenshot.png)

# Troubleshooting

If after copying the directory into the plugins folder of the TeXmacs personal initialization directory the buttons do not appear, please 
1) Update the plugins from the menu (`Tools` -> `Update` -> `Plugins`)
2) Try setting the document in `paper` mode while in the Beamer style (`Focus` -> `Slideshow` -> `Paper`).

You may need to switch between `screens` and `paper` mode, this says the forum post http://forum.texmacs.cn/t/is-there-a-nicer-way-to-reorganize-slides/868/12 .

The forum thread http://forum.texmacs.cn/t/texmacs-beamer-order-change/1531 might also contain helpful information for installation troubleshooting.
